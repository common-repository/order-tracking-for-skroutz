Track orders that came from Skroutz inside your WooCommerce orders list.

== Description ==

Track and filter orders coming to your online store via Skroutz.

Now with this plugin, you will be able to distinguish the orders that come from the Skroutz platform, through a distinctive label that you place in your order list, as well as in the detailed - full view of the order.

Please for any questions/support don't hesitate to contact us thrue plugins@frontseries.gr

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

That's it! No need for configuration.

== Frequently Asked Questions ==

= What does this plugin do? =

Checks if the order you received from your eshop comes from Skroutz

= How does it check that? =

Integrated with Skroutz Analytics Code and read the cookies in the "Thank you" (create order) state.

= Do I need to have Skroutz Analytics in my site? =

Yes. In order this plugin to work properly, you must install and setup the Skroutz Analytics javascript code.

= I have installed my Skroutz Analytics but still can't see orders from Skroutz. =

Keep in mind that if a user have an extension that blocks outside scripts such as Skroutz Analytics, the plugin cannot work properly.
